Scaleset Utils
==============

Scaleset Utils is a collection of utility clases used by the various scaleset frameworks.

Quick Start
-----------

### Dependency

```java
<dependency>
    <groupId>com.scaleset</groupId>
    <artifactId>scaleset-utils</artifactId>
    <version>0.1.0</version>
</dependency>
```
